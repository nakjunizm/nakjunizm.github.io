Map кіј Set

2017-04-21
Map, Set, java

мқҙ нҸ¬мҠӨнҠё м—җм„ңлҠ” Map кіј Set м—җ лҢҖн•ҙ м•Ңм•„ліҙлҸ„лЎқ н•ҳкІ мҠөлӢҲлӢӨ

Map (Java Platform SE 8 )
Set (Java Platform SE 8 )

# н•өмӢ¬

  • MapмқҖ Key, Value мҢҚмқ„ к°Җм§ҖлҠ” мҳӨлёҢм қнҠё мһ…лӢҲлӢӨ. KeyлҠ” мӨ‘ліөлҗ  мҲҳ м—ҶмҠөлӢҲлӢӨ.
  • SetмқҖ мҲҳн•ҷ м—җм„ңмқҳ м§‘н•©кіј лҸҷмқјн•ң к°ңл…җ мһ…лӢҲлӢӨ. мӨ‘ліөлҗҳм§Җ м•ҠлҠ” к°’мқ„ к°–лҠ” мҳӨлёҢм қнҠё мһ…лӢҲлӢӨ.
  • Mapкіј Set лӘЁл‘җ мҲңм„ңлҘј ліҙмһҘн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ

keyлҠ” Integer, valueлҠ” Stringмқё HashMapмқҳ playлқјлҠ” ліҖмҲҳлҘј л§Ңл“Өкі 
1 м—җ мЎ°мҪ”л№„м№ҳ, 2 м—җ лЁёл ҲмқҙлҘј л„Јм—ҲмҠөлӢҲлӢӨ.
лӢӨмӢң 1м—җ лЁёл ҲмқҙлҘј л„ЈкІҢлҗҳл©ҙ лӮҳмӨ‘м—җ л“Өм–ҙк°„ лЁёл Ҳмқҙк°Җ мЎ°мҪ”л№„м№ҳлҘј лҚ®м–ҙмҚЁлІ„л Өм„ң кІ°көӯ key 1, 2мқҳ valueлҠ” лӘЁл‘җ вҖңMurrayвҖқк°Җ лҗңкІғмқ„ нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.

Setм—җлҠ” к°’л§Ң л“Өм–ҙмһҲкё° л•Ңл¬ём—җ 4лӘ…мқҳ н”Ңл Ҳмқҙм–ҙмқҳ мқҙлҰ„мқ„ л„Јкі , лӢӨм„ҜлІҲм§ём—җ вҖңDjokovicвҖқмқ„ н•ңлІҲ лҚ” л„Јм—ҲмқҢм—җлҸ„ л¶Ҳкө¬н•ҳкі 
мӨ‘ліөк°’мқҖ н—Ҳмҡ©лҗҳм§Җ м•Ҡкё° л•Ңл¬ём—җ player setмқҳ sizeлҠ” м—¬м „нһҲ 4мқёкІғмқ„ м•Ң мҲҳ мһҲмҠөлӢҲлӢӨ.

мң„мқҳ мҳҲм—җм„ң мӮ¬мҡ©н•ң HashMapмқҖ Javaм—җм„ң Map InterfaceлҘј мӮ¬мҡ©н•ҳлҠ” к°ҖмһҘ мқјл°ҳм Ғмқё л°©мӢқ мһ…лӢҲлӢӨ.
ArrayList мҷҖ LinkedList +@ м—җм„ңлҸ„ мһ к№җ м–ёкёүн•ң м Ғмқҙ мһҲлҠ”лҚ° л©ҖнӢ° м“°л Ҳл“ңлЎң к°ңл°ңмқ„ н•ҳлҠ” кІҪмҡ°лҠ” н•ӯмғҒ мӮ¬мҡ©н•ҳкі мһҗ н•ҳлҠ” нҒҙлһҳмҠӨлӮҳ мқён„°нҺҳмқҙмҠӨк°Җ Thread safe н•ңм§Җ мӮҙнҺҙ лҙҗм•ј н•©лӢҲлӢӨ.

HashMapмқҖ лҸҷкё°нҷ” лҗҳм–ҙмһҲм§Җ м•Ҡкё° л•Ңл¬ём—җ лҸҷмӢңм—җ м—¬лҹ¬к°ңмқҳ м“°л Ҳл“ңк°Җ лҸҷмӢңм—җ к°’мқ„ мҲҳм •н•  кІҪмҡ° мқөм…үм…ҳмқҙ л°ңмғқн•  мҲҳ мһҲмңјлӢҲ н•„мҡ”н•ң кІҪмҡ° Collections.synchronizedMap н•ЁмҲҳлҘј мӮ¬мҡ©н•ҳкұ°лӮҳ ConcurrentHashMapмқ„ мӮ¬мҡ©н•ҳм—¬ мқҙ л¬ём ңлҘј нҡҢн”јн•  мҲҳ мһҲмҠөлӢҲлӢӨ.
(JDK1.5 л¶Җн„° л“ұмһҘн•ң ConcurrentHashMapмқ„ мӮ¬мҡ©н•ҳлҠ” нҺёмқҙ лҚ” нҡЁмңЁм Ғ мһ…лӢҲлӢӨ.)

# HashMapм—җм„ң HashлҠ” л¬ҙмҠЁ м—ӯн• мқ„ н•ҳлҠ”к°Җ

л„ӨмқҙлІ„мқҳ helloworld лё”лЎңк·ём—җ мһҗм„ён•ң лҸҷмһ‘л°©мӢқмқҙ м„ӨлӘ…лҗҳм–ҙ мһҲмңјлӢҲ к¶ҒкёҲн•ҳмӢ л¶„л“ӨмқҖ н•ңлІҲм”© мқҪм–ҙліҙмӢңл©ҙ лҸ„мӣҖмқҙ лҗ кІғ к°ҷмҠөлӢҲлӢӨ.

мҡ”м•Ҫн•ҙм„ң мүҪкІҢ м„ӨлӘ…мқ„ н•ҙ ліҙл©ҙ

HashMapмқҖ key-value нҺҳм–ҙлҘј лӢҙмқ„мҲҳ мһҲлҠ” bucketsмқ„ к°Җм§Җкі  мһҲмҠөлӢҲлӢӨ.
put(key,value)л©”мҶҢл“ңлҘј нҳём¶ңн•ҳл©ҙ к°Ғ key-value нҺҳм–ҙк°Җ м–ҙлҠҗ bucketм—җ л“Өм–ҙк°Ҳм§Җ кІ°м •н• л•Ң л°”лЎң hashCode()н•ЁмҲҳк°Җ мӮ¬мҡ© лҗ©лӢҲлӢӨ.

get(key)лҘј нҳём¶ңн•ң кІҪмҡ°м—җлҸ„ м–ҙлҠҗ bucketм—җм„ң лҚ°мқҙн„°лҘј к°Җм ёмҳ¬м§Җ кІ°м •н• л•Ң hashCode()к°Җ мӮ¬мҡ© лҗ©лӢҲлӢӨ.
(лӮҙл¶Җм ҒмңјлЎң мқҙ hashcodeмқҳ лҰ¬н„ҙк°’мңјлЎң shift м—°мӮ°мқ„ нҶөн•ҙ кІ°м •лҗңлӢӨкі  н•ҳл„Өмҡ”)

м–ҙлҠҗ bucketмқём§Җ м•Ңм•ҳмңјл©ҙ мӢӨм ң к·ёлІ„нӮ·м—җ л“Өм–ҙмһҲлҠ” лӘЁл“  нӮӨмқҳ hashcode к°’лҒјлҰ¬ equals()лҘј нҶөн•ҙ мӣҗн•ҳлҠ” putмқҙлӮҳ get мқҳ кІ°кіјлҘј м–»мқ„ мҲҳ мһҲкІҢ лҗ©лӢҲлӢӨ.

# +@ HashMap VS HashTable

  1. HashtableмқҖ synchronized, HashMap мқҖ м•„лӢҳ
  2. Hashtable мқҖ nullмқ„ keyлӮҳ valueлЎң к°Җм§ҲмҲҳ м—ҶмқҢ, HashmapмқҖ н•ҳлӮҳмқҳ null keyмҷҖ м—¬лҹ¬к°ңмқҳ null к°’мқ„ к°Җм§Ҳ мҲҳ мһҲмқҢ
  3. Hashmapмқ„ мӮ¬мҡ©н•ҳлӢӨк°Җ мҲңм„ңк°Җ мӨ‘мҡ”н•ҙ м§„ кІҪмҡ° лӢЁмҲңнһҲ Hashmapмқҳ м„ м–ёл¶ҖлҘј LinkedHashMapмңјлЎң ліҖкІҪн•ҙмЈјл©ҙ лӮҳлЁём§Җ мҪ”л“ңлҠ” кұҙл“ңлҰҙ н•„мҡ”к°Җ м—ҶмңјлӮҳ HashtableлЎң н•ҳл Өл©ҙ кіЁм№ҳк°Җ м•„нҢҢм§җ.
